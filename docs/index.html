Install onePiece-plymouth â€” helper

1. Clone the repo (to /usr/share/plymouth/themes) 
sudo git clone https://github.com/Anxhul10/onePiece-plymouth.git /usr/share/plymouth/themes/onePiece-plymouth
2. Register the theme with update-alternatives 
sudo update-alternatives --install /usr/share/plymouth/themes/default.plymouth default.plymouth /usr/share/plymouth/themes/onePiece-plymouth/onePiece-plymouth.plymouth 120
3. Select the default theme (interactive) 
sudo update-alternatives --config default.plymouth
4. Update initramfs 
sudo update-initramfs -u
5. Reboot 
sudo reboot
Note for Ubuntu < 16.04
